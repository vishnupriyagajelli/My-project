# My-project
# Television Market Analysis – Flipkart Web Scraping & EDA

##  Project Overview
This project focuses on analyzing the Indian television market by scraping product data from Flipkart.  
The goal is to understand pricing trends, popular brands, and customer preferences using data analysis techniques.

---

## Tools & Technologies Used
- Python
- BeautifulSoup / Requests (Web Scraping)
- Pandas & NumPy (Data Cleaning)
- Matplotlib / Seaborn (Visualization)
- Jupyter Notebook

---

##  Key Analysis Performed
- Scraped television product data from Flipkart
- Cleaned and preprocessed raw data
- Analyzed price distribution and brand-wise trends
- Identified top brands and preferred screen sizes
- Visualized insights using charts and graphs

---

##  Key Insights
- Identified popular TV brands based on availability and pricing
- Observed pricing trends across different screen sizes
- Highlighted customer preferences using ratings and reviews
